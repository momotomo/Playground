# MVP_SCOPE

## 今回実装した範囲

- `egui + eframe` ベースの単一 crate 構成
- native / web の共通アプリ本体
- 左側ツールパネル、上部基本操作、中央キャンバスの UI
- マウスドラッグによる自由線描画
- 色変更
- 線幅変更
- 消しゴムモード
- `Undo`
- `Redo`
- `Clear`
- 編集用 JSON 形式での保存 / 読込
- native の実ファイル保存 / 読込
- web のダウンロード保存 / ファイル選択読込
- 読込失敗時のエラーメッセージ
- `Trunk` による wasm ローカル起動基盤
- GitHub Pages 配布用 workflow
- 保存形式 / 履歴管理のテスト
