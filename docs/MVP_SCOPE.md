# MVP_SCOPE

## MVP 完成としての到達点

- GitHub Pages でそのまま触れる web 版と、同じコードベースの native 版を用意
- `Save JSON` による再編集用保存と、`Export PNG` による共有用出力を分離
- 描画、選択、複数選択、変形、整列、レイヤー、補助線までを含む「公開して触れる MVP」を成立
- 最小ヘルプ、空状態、公開向け文言を整え、初見でも少し触れば役割が分かる状態にする
- キーボードなしでも最低限使えるタブレット向け導線を用意する
- タブレットでもかなり自然に触れるズーム / パン / 選択導線を用意する
- タブレットでの描画とビュー操作の役割分離を、可能な範囲で実用的に整える
- 小さい画面でも左サイドバーを最後まで操作できる状態にする
