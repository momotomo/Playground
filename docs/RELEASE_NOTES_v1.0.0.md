# v1.0.0 - 初回公開版

このリリースは、このツールの初回公開版です。

Rust / egui / eframe で作った、初心者でも迷いにくく使える高性能寄りのお絵かきツールとして、一旦完成扱いの区切りにしています。

公開 URL:
[https://momotomo.github.io/Playground/](https://momotomo.github.io/Playground/)

主な機能:

- freehand、消しゴム
- 矩形 / 楕円 / 直線
- 単一選択 / 複数選択
- 移動 / リサイズ / 回転
- Group / Ungroup
- 整列 / 等間隔配置 / 重なり順変更
- レイヤー
- JSON 保存 / 読込
- PNG / 透過PNG / SVG 書き出し
- グリッド / ガイド / スナップ
- スマートガイド
- ルーラー
- 日本語 UI
- タブレット対応
- ブラシ 4 種:
  `ペン / えんぴつ / マーカー / クレヨン`

公開方針:

- GitHub Pages で Web 版を公開しています
- ソースコードは MIT License で公開しています
- 現時点では一旦完成扱いですが、必要に応じて今後も改善していきます

補足:

- JSON は再編集用です
- PNG / 透過PNG は見たまま共有向けです
- SVG は図形や線の再利用向けです
