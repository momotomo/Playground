# PROJECT_CONTEXT

## 目的

- Rust だけで動く、お絵かきと軽い編集ができるツールを育てる
- `native` と `wasm` を同じコードベースで維持する
- 初見でも迷いにくい UI を保ちながら、段階的に高機能化する

## 技術スタック

- Rust
- `egui`
- `eframe`
- native / `wasm32-unknown-unknown`
- GitHub Pages で Web 公開

## 保存と export

- 再編集用: JSON
  - 現在の保存形式は `format.version = 4`
  - 既存コードでは旧 `v1 / v2 / v3` 読込互換を持つ
- 共有用: PNG
  - `PNG書き出し`: 見たまま共有向け
  - `透過PNG`: 透明素材向け
- 再利用向け: SVG
  - 図形と線を中心に持ち出す
  - freehand やバケツ塗りは安全側で簡略化する
  - freehand は軽い簡略化と smoothing をかけた path として出力する

## UX 方針

- 日本語 UI 前提
- 初心者でも迷いにくい
- 高機能化しても常設長文説明だらけにしない
- タブレットでの押しやすさと状態把握を重視する
- マウス / キーボード / タッチ / ペンの共存を保つ

## 現在の主要実装

- フリーハンド描画: `ペン / えんぴつ / マーカー / 消しゴム`
- 図形: `直線 / 四角形 / 楕円`
- 色: 線色 / 塗り色 分離、不透明度、最近使った色、簡易パレット、スポイト
- 塗り: 図形の塗り、`バケツ塗り`、5段階の `塗りのゆるさ`
- 編集: 単一選択 / 複数選択、移動、リサイズ、回転、整列、等間隔配置、重なり順、グループ化、単一図形と複数図形の style 編集、freehand / fill-only 選択の一括 style 編集
- レイヤー: 追加 / 複製 / 削除 / 名前変更 / 表示 / ロック / 並び替え / レイヤー間移動
- 補助表示: グリッド、ガイド、スマートガイド、ルーラー
- Web / native 共通の save/load、PNG / 透過PNG / SVG export

## 設計上の重要前提

- 作品状態、ビュー状態、選択状態は分離する
- 選択や UI 補助表示は export に含めない
- `FillElement` はバケツ塗り結果を保持する第一版の塗り要素
- visible な全レイヤーの見た目を塗り境界判定に使い、結果は作業レイヤーへ入れる
- SVG は安全優先で簡略化する。見たまま共有は PNG 系の役割に残す

## 壊してはいけないもの

- PNG / 透過PNG / SVG export
- render
- selection / transform
- layer
- fill
- wasm / GitHub Pages
- 日本語フォント埋め込み前提
