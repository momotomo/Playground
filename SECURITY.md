# セキュリティについて

このプロジェクトに関するセキュリティ上の問題や、脆弱性の可能性がある挙動を見つけた場合は、公開 issue に詳細をすぐ書かず、まずは GitHub 上の連絡手段で知らせてください。

個人開発ベースのため、即時対応を約束することはできません。  
ただし、確認でき次第、内容を確認し、できる範囲で対応します。

## お願い

- 再現手順がある場合は、分かる範囲で整理してください
- 影響範囲が分かる場合は、あわせて共有してください
- 公開前提の GitHub Pages / WebAssembly アプリであることを踏まえつつ、必要以上に公開拡散せず、まずは連絡をお願いします

## このプロジェクトの構成について

このツールは主に次の前提で動作します。

- GitHub Pages 上で公開される WebAssembly アプリ
- ローカル保存中心
- サーバーを持たない構成

そのため、一般的な Web サービスとはセキュリティ上の論点が異なる場合がありますが、問題を見つけた際は遠慮なく知らせてください。
