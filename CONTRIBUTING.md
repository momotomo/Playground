# コントリビューションについて

このプロジェクトに興味を持っていただきありがとうございます。

このツールは、初心者でも迷いにくく使える高性能寄りのお絵かきツールを目指して育ててきたもので、現時点では一旦完成扱いにしています。  
そのため、今後の変更は大きな作り直しよりも、使い勝手の改善、不具合修正、小さな磨き込みを優先したいと考えています。

## 歓迎するもの

- 不具合報告
- 表示や文言の改善提案
- 使い勝手を良くする小さな修正
- docs / README / ヘルプの改善
- Web / wasm / タブレット利用時の違和感報告
- 既存方針を尊重した小さな実装改善

## 事前相談があるとうれしいもの

- 大きな UI 方針変更
- ツール体系の再設計
- 保存形式への影響がある変更
- JSON v4 互換に関わる変更
- 初心者向け UX を大きく変える変更
- wasm / GitHub Pages 公開に影響する変更

## このプロジェクトで特に大事にしていること

- 日本語 UI として自然であること
- 初見の人でも迷いにくいこと
- 常設の長文説明に頼りすぎないこと
- Web / native / wasm の両立
- JSON v4 の互換性
- save/load、PNG / 透過PNG / SVG export の整合を壊さないこと

## 改善を提案するときの目安

まずは issue で相談してもらえるとうれしいです。  
特に大きな変更は、実装前に方向性を共有してもらえると助かります。

コード変更を行う場合は、既存実装を尊重し、できるだけ最小変更で目的を達成してください。

ありがとうございます。
